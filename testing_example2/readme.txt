This project is for showing how to build a good project directory structure
Needs good knowledge of modules and running python in 'module mode' e.g.
python -m .....

Run main
--------
python main.py

How to run a script
--------------------
python -m scripts.util1

How to run all tests
--------------------
python -m unittest discover tests -p '*.py'

Run just a library module
-------------------------
python -m lib.light_calculator
